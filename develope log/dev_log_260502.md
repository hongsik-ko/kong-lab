## 3. 직렬화(Serialization) 이슈 해결

### 문제
- Redis 캐시 저장 시 에러 발생
- `Cannot serialize`
- `JdkSerializationRedisSerializer` 관련 오류

### 원인
- 기본 JDK 직렬화 방식 사용
- DTO 구조와 호환되지 않음

### 해결
- JSON 직렬화 방식으로 변경
- `GenericJackson2JsonRedisSerializer` 적용
