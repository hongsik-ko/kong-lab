- AI 활용: ChatGPT + Cursor

# 🏗️ 공통 설계

## BasicEntity
- createdAt (UTC)
- createdBy
- modifiedAt (UTC)
- modifiedBy
- recordStatus (1자리 enum)

## RecordStatus
- N: Normal
- D: Deleted
- H: Hidden
- E: Error

## Audit
- @CreatedDate / @LastModifiedDate
- @CreatedBy / @LastModifiedBy
- AuditorAware로 사용자 자동 주입

## 임시 로그인 구조
- SessionUserContext
- SessionUserProvider
- userId = 1L (하드코딩)
